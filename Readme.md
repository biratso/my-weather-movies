# My Weather Movies

A project that combines weather information with movie recommendations.

## Description

This application provides movie recommendations based on current weather conditions.

## Installation

```bash
npm install
```

## Usage

```bash
npm start

npm run dev green

# Build any variant, any platform
npm run build android -- --profile green
npm run build ios -- --profile orange
npm run build all -- --profile green

# Publish update to Green variant
npm run update:green "Fixed login bug"

# Publish update to Orange variant
npm run update:orange "Added new feature"

```



## Features

- Weather data integration
- Movie recommendations
- User-friendly interface

## Technologies Used

- Node.js
- Weather API
- Movie Database API
